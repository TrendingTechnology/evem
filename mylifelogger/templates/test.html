What is mylifelogger?

My lifelogger is a simple command line application for sending reminders for events through email, like this email you are currently getting.

How it works?

User can:

  • Create new events
  • Set reminders for those events
  • Edit, delete and look at those events
  • It is designed such that if this script is run on the startup, it checks all events and checks if there is any event which reminder date is today.
  • If so it sends an HTML email as reminder
